Sry no support for NoClip in AS because it won't help you much.
On the serverside your position is validated which does not allow you to pass through everything.
Simple gameobjects like teleporter statues in main cities are client side only but for example doors/walls are serverside based.
This is also the cause for teleporting. You will get backported because the serverside says no, thats new position from the old position is not possible. The same applies to run speed, if you increase it too high you will get backported.
