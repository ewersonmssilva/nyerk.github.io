Hi, I saw people were asking about sandboxie and how to run more than 2 clients. I already explained that I personally had to purchase the sandboxie because windows 10 didn't have a cracked version but that also saved me a ton of trial and error. The sandboxie program is paid once and it has a lifetime license so you will never have to pay for it again even if there is an update or a new version.

So I made some photos below to show you some steps. If any photo is not clearly visible this is because the image is shrunk in order to fit the forum, in this case simply click the link I provide above each photo so as to open it in a new window and see it full size

Making a sandboxie of yourself is easy via the menu:

https://s10.postimg.org/jpg85qdzd/image.jpg


then you will get his window where you simply have to name it and choose whether you want anything to be copied by another folder, hit "none" on this one you won't need it.

https://s10.postimg.org/h84gyijt5/image.jpg


Then you will see your new sandboxie in the list of available ones, it will be sorted according to name so name them like you need so they are "grouped" for an easy access.

https://s10.postimg.org/f3k3xhv21/image.jpg


##########################################################

Now how to use Sandboxie with Aion the first time.

https://s10.postimg.org/3r7ifvu4p/image.jpg


Make the exact amount of sandboxies to the amount of accounts you want to play simultaneously. You want to open one client per sandboxie so the client will open instantly.

https://s10.postimg.org/k3hk5a8cp/image.jpg


When you want to update the client or if you have a client crash in a sandboxie all you have to do is Delete contents and start fresh.

https://s10.postimg.org/nac3p3fzt/image.jpg
