How to launch 32-bit on EU

Posted: Thu Jan 12, 2017 7:52 pm
by agonic
  • Download the ZIP archive and extract it
  • Move the aion_l.bat file to your Aion main folder (usually Download - you should see bin32 and bin64). DO NOT place it inside any bin folder!
  • You can run Aion's 32 Bit version by executing the BAT file
  • Right click the .BAT file and create a shortcut for the desktop.
  • If you want to auto login, edit the aion_l.bat file and add following parameters -:account:your account name, -password:password of your account.
