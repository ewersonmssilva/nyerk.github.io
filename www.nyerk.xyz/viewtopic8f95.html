It does have three movement types:

MoveTo which does the spastic mini teleports -> dead giveaway you bot, only used inside instances
MoveSmooth which moves just like you would between points
FlySmooth a recent addition that was added after I told Paraly that the flight script was bad because it was using mini teleports, this flies like you would fly yourself.

Another thing it does is allows you to make it put the pin automatically which needs the client to be on foreground momentarily while it recognizes the letters.

But there is nothing to put your username and password, because you need to be already logged in with a char to start Vanilla already. The only thing you would need pin for is if one or more of your chars to be logged in an account are in FT server or go to Panesterra.
