Demn what is so hard to do?

1. Unzip the bin folder and copy it to your aion folder wherre there is bin32 and bin64 folder.

2. Run aion normally,

3. When aion loading window starts, run RebootAion32CustomBin, this program will kill your aion process and start another process loading aion from the costum bin folder that u pasted from step 1, witch contains the aion.bin file that runs with AionScript. Do not let the loading bar load to 100%, just run RebootAion32CustomBin when you get the aion client frame opened, id say run it after 2 seconds.

4. After this, when you are ingame u can run AionScript.

Post by Diocan » Thu Mar 29, 2018 10:12 am

1. Put bin folder in aion installation folder.

2. Run aion in 64 bit version. Make sure you launch aion.bin from bin64 folder.

3. Run RebootAionCostumBin as soon as you see the loading page of aion ( 2 seconds after you launched aion ). This will kill your aion 64bit process and start aion from the bin folder that have the right offsets to make AionScript work.

4. Run aionscript once you are logged into the game
