How to use Aidpsmeter and Aionrainmeter

  • Quote

Post by agonic » Wed Feb 01, 2017 8:54 am

Both of the softwares aim that measure the dps and some specific conditions.

AIDPSMETER; http://aioninfo.com/dpsmeter.html
  • Captures data packs to measure values
  • Measure dps and healing
  • Was showing NoAnimation usage; not atm for now
  • Worst feature is to must run aiDPSMeter before login to aion server and must install Win10pcap file
  • Best works with Ethernet connection
  • Never supports USB stick connections
AIONRAINMETER (ARM); http://rainy.ws/download/
  • Scans whole chat lines to measure/show values
  • Have multiple features
  • Can use whenever you want; before or after login to aion server
  • Can take SS
  • You can translate between asmo-ely
  • You can add triggers to warn you
  • You can search whole chat lines
  • You can make player database
  • Under Help, there is a section that Tips/Help to use it via some chat commands
etc etc etc.. Install and see

Using chat commands;
  • Write commands as you see in the Tips/Help
  • Ctrl+A and then Ctrl+X your commands. Tataa, It works
Trigger function of ARM;
  • It is trigger for spesific word/words in the whole chat lines to warn you via popup or soundfile
  • When It detects your trigger, It makes a tray popup or play MP3 or etc
examples;
  • Add Whispers to warn you when you get a whisper
  • Add silenced you to warn you when you get silenced
  • Or You WTB something, then add it as trigger to warn you when it is exist in LFG or etc
That's all for now. If you have any question, Don't hesitate to ask
