Getting started

Posted: Thu Jan 12, 2017 7:59 pm
by agonic
  • Download Aionscript
  • If you are a NA Player you also need Reboot Aion32, you can find it here
  • To run AionScript you also need to download the following Software Microsoft .NET Framework 4.6.2 and Microsoft Visual C++ 2010 Redistributable Package (x86)
  • Start Aion in 32 Bit mode EU Players and for NA Players
  • After you entered the world (After Character Select), start AionScript
  • Be sure that AionScript.exe is started with Administration Rights. (Right Click and select "Run as administrator")
  • If you don't see the Entity and Radar Windows then AionScript needs an Update or you didn't start with Administrator Rights.
