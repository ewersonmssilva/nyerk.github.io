1. Put bin folder in aion installation folder.

2. Run aion in 64 bit version. Make sure you launch aion.bin from bin64 folder.

3. Run RebootAionCostumBin as soon as you see the loading page of aion ( 2 seconds after you launched aion ). This will kill your aion 64bit process and start aion from the bin folder that have the right offsets to make AionScript work.

4. Run aionscript once you are logged into the game
