How to launch 32-bit on NA without bat

Post by agonic » Thu Jan 12, 2017 7:45 pm

RebootAion32 is a tool to force the NA Launcher to start the 32Bit Version of Aion.
If you have a 64 Bit System, NA forces you to use the 64Bit Version of Aion.

This Tool will remove the restriction and starts instead of the 64 Bit Client the 32 Bit Client.
  • Download RebootAion32
  • Launch RebootAion32
  • Start Aion via Launcher
